# 活动管理 - 删除活动模块

## 测试点

### P0 - 删除权限（按状态）

- TC-ACTIVITY-DELETE-001: "活动待提交"状态的活动可删除
  预期结果：详情页显示【删除】按钮

- TC-ACTIVITY-DELETE-002: "活动已发布"状态的活动不可删除
  预期结果：详情页不显示【删除】按钮

- TC-ACTIVITY-DELETE-003: "活动已完成"状态的活动不可删除
  预期结果：详情页不显示【删除】按钮

- TC-ACTIVITY-DELETE-004: "活动已取消"状态的活动不可删除
  预期结果：详情页不显示【删除】按钮

### P0 - 数据权限

- TC-ACTIVITY-DELETE-005: 活动创建者可以删除自己的待提交活动
  预期结果：成功删除，列表不再显示

- TC-ACTIVITY-DELETE-006: 非创建者无法删除他人的待提交活动
  预期结果：删除按钮不可见或点击后提示无权限

### P0 - 删除流程

- TC-ACTIVITY-DELETE-007: 点击【删除】按钮
  预期结果：弹出确认删除弹框

- TC-ACTIVITY-DELETE-008: 确认删除
  预期结果：跳转列表页，显示"活动删除成功"toast，活动不在列表显示

- TC-ACTIVITY-DELETE-009: 取消删除
  预期结果：弹框关闭，活动未被删除

### P2 - 操作历史

- TC-ACTIVITY-DELETE-010: 删除后系统留存操作记录
  预期结果：操作历史中有删除记录（页面不可见，后台可查）
