# 活动管理 - 文件上传模块

## 测试点

### P0 - 支持格式

- TC-ACTIVITY-FILE-001: 上传.pdf文件
  预期结果：上传成功，显示在文件列表中

- TC-ACTIVITY-FILE-002: 上传.jpg文件
  预期结果：上传成功，显示在文件列表中

- TC-ACTIVITY-FILE-003: 上传.png文件
  预期结果：上传成功

- TC-ACTIVITY-FILE-004: 上传.gif文件
  预期结果：上传成功

- TC-ACTIVITY-FILE-005: 上传.xls文件
  预期结果：上传成功

- TC-ACTIVITY-FILE-006: 上传.xlsx文件
  预期结果：上传成功

### P0 - 不支持格式

- TC-ACTIVITY-FILE-007: 上传.exe文件
  预期结果：提示"不支持的文件格式"，不上传

- TC-ACTIVITY-FILE-008: 上传.zip文件
  预期结果：提示"不支持的文件格式"，不上传

### P0 - 大小限制

- TC-ACTIVITY-FILE-009: 上传单个文件大小超过20MB
  预期结果：提示"文件大小超出限制"，不上传

- TC-ACTIVITY-FILE-010: 上传多个文件总大小超过20MB
  预期结果：提示"总文件大小超出限制"

- TC-ACTIVITY-FILE-011: 上传多个文件总大小在20MB以内
  预期结果：全部上传成功

### P1 - 删除文件

- TC-ACTIVITY-FILE-012: 上传文件后点击删除按钮
  预期结果：文件从列表中移除

- TC-ACTIVITY-FILE-013: 上传多个文件后删除其中一个
  预期结果：只删除选中的文件，其他保留

### P2 - 文件数量

- TC-ACTIVITY-FILE-014: 上传超过数量限制的文件
  预期结果：提示超出数量限制

- TC-ACTIVITY-FILE-015: 详情页可预览/下载文件
  预期结果：点击文件链接可预览或下载
