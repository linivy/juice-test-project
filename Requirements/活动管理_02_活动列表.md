# 活动管理 - 活动列表模块

## 测试点

### P0 - 数据权限

- TC-ACTIVITY-LIST-001: Admin账号查看活动列表
  预期结果：显示系统中所有正式提交的活动 + 自己保存的草稿

- TC-ACTIVITY-LIST-002: 普通角色查看活动列表
  预期结果：只显示自己正式提交的活动 + 自己保存的草稿

- TC-ACTIVITY-LIST-003: 已删除的活动不显示在列表中
  预期结果：删除活动后，该活动不在列表中出现

### P0 - 列表字段显示

- TC-ACTIVITY-LIST-004: 列表显示所有配置字段
  预期结果：活动类型、活动名称、活动地点、开始时间、结束时间、活动状态、操作

- TC-ACTIVITY-LIST-005: 活动名称超长（超过两行）
  预期结果：显示"..."截断，鼠标悬停显示完整名称

- TC-ACTIVITY-LIST-006: 活动地点格式正确
  预期结果：格式为"线上/线下，{城市，具体地点}"

### P1 - 搜索功能

- TC-ACTIVITY-LIST-007: 按活动名称精确搜索
  预期结果：列表只显示匹配的活动

- TC-ACTIVITY-LIST-008: 按活动名称模糊搜索
  预期结果：列表显示包含关键词的所有活动

- TC-ACTIVITY-LIST-009: 搜索不存在的活动名称
  预期结果：显示"暂无数据"空状态提示

- TC-ACTIVITY-LIST-010: 清空搜索条件
  预期结果：恢复显示全部数据

### P1 - 筛选功能

- TC-ACTIVITY-LIST-011: 按活动类型筛选（级联筛选）
  预期结果：一级选择后，二级联动

- TC-ACTIVITY-LIST-012: 按活动状态筛选（单选/多选）
  预期结果：只显示选定状态的活动

- TC-ACTIVITY-LIST-013: 按活动地点筛选（线上/线下）
  预期结果：只显示选定地点的活动

- TC-ACTIVITY-LIST-014: 按自定义时间段筛选
  预期结果：只显示时间范围内的活动

- TC-ACTIVITY-LIST-015: 多条件组合筛选
  预期结果：同时满足所有筛选条件的活动

- TC-ACTIVITY-LIST-016: 清除所有筛选条件
  预期结果：恢复显示全部数据

### P1 - 排序功能

- TC-ACTIVITY-LIST-017: 默认排序（按更新时间倒序）
  预期结果：最新更新的活动排在最前面

- TC-ACTIVITY-LIST-018: 点击"活动开始时间"表头排序
  预期结果：第一次点击正序，第二次点击倒序

- TC-ACTIVITY-LIST-019: 点击"活动结束时间"表头排序
  预期结果：第一次点击正序，第二次点击倒序

### P1 - 分页功能

- TC-ACTIVITY-LIST-020: 每页显示10条数据
  预期结果：超过10条时显示分页组件，每页最多10条

- TC-ACTIVITY-LIST-021: 点击下一页
  预期结果：跳转到第二页，显示剩余数据

- TC-ACTIVITY-LIST-022: 点击上一页
  预期结果：返回上一页

- TC-ACTIVITY-LIST-023: 点击末页/首页
  预期结果：跳转到最后一页/第一页

- TC-ACTIVITY-LIST-024: 输入页码跳转
  预期结果：跳转到指定页

- TC-ACTIVITY-LIST-025: 数据不足10条时
  预期结果：不显示分页组件，或显示总页数为1

### P2 - 导出功能

- TC-ACTIVITY-LIST-026: 点击导出按钮
  预期结果：导出当前列表数据（格式为.xlsx或.csv）

- TC-ACTIVITY-LIST-027: 无数据时点击导出
  预期结果：提示"没有可导出的数据"
