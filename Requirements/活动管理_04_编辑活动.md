# 活动管理 - 编辑活动模块

## 测试点

### P0 - 编辑权限（按状态）

- TC-ACTIVITY-EDIT-001: "活动待提交"状态的活动可编辑
  预期结果：详情页显示【编辑】按钮，点击进入编辑模式

- TC-ACTIVITY-EDIT-002: "活动已发布"状态的活动可编辑
  预期结果：详情页显示【编辑】按钮，点击进入编辑模式

- TC-ACTIVITY-EDIT-003: "活动已完成"状态的活动不可编辑
  预期结果：详情页不显示【编辑】按钮

- TC-ACTIVITY-EDIT-004: "活动已取消"状态的活动不可编辑
  预期结果：详情页不显示【编辑】按钮

### P0 - 数据权限

- TC-ACTIVITY-EDIT-005: Admin可编辑所有活动
  预期结果：Admin能看到所有活动的编辑按钮并成功编辑

- TC-ACTIVITY-EDIT-006: 普通用户只能编辑自己创建的活动
  预期结果：只能看到和编辑自己的活动，无法看到他人的编辑按钮

### P0 - 编辑流程

- TC-ACTIVITY-EDIT-007: 点击【编辑】按钮进入编辑模式
  预期结果：表单预填当前活动的所有数据

- TC-ACTIVITY-EDIT-008: 修改活动名称后保存（草稿状态）
  预期结果：保存成功，列表页显示新名称

- TC-ACTIVITY-EDIT-009: 修改所有字段后提交（已发布状态）
  预期结果：显示"活动编辑成功"toast，操作历史新增"编辑"记录

- TC-ACTIVITY-EDIT-010: 编辑后再次保存草稿
  预期结果：活动状态保持"活动待提交"

### P1 - 编辑时的校验

- TC-ACTIVITY-EDIT-011: 编辑时清空必填字段后提交
  预期结果：提示必填字段错误，不保存

- TC-ACTIVITY-EDIT-012: 编辑时修改活动时间为过去时间
  预期结果：提示"活动时间不可选历史时间"

- TC-ACTIVITY-EDIT-013: 编辑时修改活动时长超过72小时
  预期结果：提示"活动时长不能超过72小时"

### P2 - 边界值

- TC-ACTIVITY-EDIT-014: 编辑时活动名称输入50字符
  预期结果：可正常保存

- TC-ACTIVITY-EDIT-015: 编辑时活动名称输入51字符
  预期结果：无法输入或提示超出限制
