# 活动管理 - 安全测试模块

## 测试点

### P0 - XSS防护

- TC-ACTIVITY-SEC-001: 活动名称输入XSS payload（<script>alert('xss')</script>）
  预期结果：脚本不执行，被转义显示

- TC-ACTIVITY-SEC-002: 活动简介输入XSS payload（<img src=x onerror=alert(1)>）
  预期结果：脚本不执行，被转义显示

- TC-ACTIVITY-SEC-003: 富文本编辑器插入XSS链接（javascript:alert(1)）
  预期结果：链接被过滤或转义，不执行

- TC-ACTIVITY-SEC-004: 搜索框输入XSS payload
  预期结果：页面不执行脚本，搜索结果正常显示或为空

### P0 - SQL注入防护

- TC-ACTIVITY-SEC-005: 活动名称输入SQL注入payload（' OR '1'='1）
  预期结果：系统不报错，正常处理

- TC-ACTIVITY-SEC-006: 搜索框输入SQL注入payload（'; DROP TABLE activities; --）
  预期结果：系统不报错，不执行恶意SQL

### P0 - 重复提交

- TC-ACTIVITY-SEC-007: 快速连续点击【完成创建】按钮多次
  预期结果：只创建一个活动，不产生重复记录

- TC-ACTIVITY-SEC-008: 快速连续点击【保存】按钮多次
  预期结果：只保存一次草稿

### P1 - 越权操作

- TC-ACTIVITY-SEC-009: 普通用户通过URL直接访问其他用户的活动详情页
  预期结果：提示无权限或跳转列表页

- TC-ACTIVITY-SEC-010: 普通用户尝试编辑其他用户的活动
  预期结果：无编辑按钮或点击后提示无权限

- TC-ACTIVITY-SEC-011: 普通用户尝试删除其他用户的待提交活动
  预期结果：无删除按钮或点击后提示无权限

### P2 - 会话管理

- TC-ACTIVITY-SEC-012: 会话超时后操作
  预期结果：提示重新登录或跳转登录页
