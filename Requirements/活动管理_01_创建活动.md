# 活动管理 - 创建活动模块

## 测试点

### P0 - 正向流程

- TC-ACTIVITY-CREATE-001: Admin角色，填写所有必填字段（活动名称、活动类型、活动时间、活动简介），选择线上地点，上传附件，点击【完成创建】
  预期结果：弹出确认弹框，确认后跳转列表页，显示"活动创建成功"toast

- TC-ACTIVITY-CREATE-002: 普通角色，填写所有必填字段，选择线下地点（省市区+详细地址），点击【保存】
  预期结果：保存为"活动待提交"状态，跳转列表页，显示"草稿已保存，稍后可以继续编辑"toast

- TC-ACTIVITY-CREATE-003: 仅填写活动名称，点击【保存】
  预期结果：保存为"活动待提交"状态，跳转列表页，显示"草稿已保存"toast

### P1 - 必填字段校验

- TC-ACTIVITY-CREATE-004: 所有必填字段为空，点击【完成创建】
  预期结果：提示“请输入活动名称”（页面验证顺序：活动名称 → 活动类型 → 开始时间 → 结束时间 → 活动简介）

- TC-ACTIVITY-CREATE-005: 仅活动名称为空，其他必填字段完整，点击【完成创建】
  预期结果：提示"请输入活动名称"，活动名称字段显示错误"请输入活动名称"

- TC-ACTIVITY-CREATE-006: 仅活动类型为空，其他必填字段完整，点击【完成创建】
  预期结果：提示"请选择活动类型"，活动类型字段显示错误"请选择活动类型"

- TC-ACTIVITY-CREATE-007: 仅活动时间为空，其他必填字段完整，点击【完成创建】
  预期结果：提示"选择开始时间"，开始时间字段显示错误"选择开始时间"

- TC-ACTIVITY-CREATE-008: 仅活动简介为空，其他必填字段完整，点击【完成创建】
  预期结果：提示"请输入活动简介"，活动简介字段显示错误"请输入活动简介"

### P0 - 级联联动

- TC-ACTIVITY-CREATE-009: 选择活动类型"社区活动"
  预期结果：二级下拉框显示"运动会"/"知识讲座"/"才艺比赛"

- TC-ACTIVITY-CREATE-010: 选择活动类型"家庭活动"
  预期结果：二级下拉框显示"亲子活动"/"户外露营"/"亲子烘焙"

- TC-ACTIVITY-CREATE-011: 选择活动类型"其他"
  预期结果：二级下拉框消失，显示"活动类型说明"文本输入框（必填，限50字符）

- TC-ACTIVITY-CREATE-012: 选择活动地点"线上"
  预期结果：显示"平台名称"输入框（限50字符），省市区地址隐藏

- TC-ACTIVITY-CREATE-013: 选择活动地点"线下"
  预期结果：显示省市区下拉框 + "详细地址"输入框（限50字符），平台名称隐藏

### P1 - 放弃创建

- TC-ACTIVITY-CREATE-014: 填写部分信息后，点击【放弃创建】，确认放弃
  预期结果：返回列表页，填写的信息不保存

- TC-ACTIVITY-CREATE-015: 填写部分信息后，点击【放弃创建】，取消放弃
  预期结果：停留在创建弹框上，已填写的信息保留

### P2 - 其他校验

- TC-ACTIVITY-CREATE-017: 重复快速点击【完成创建】按钮
  预期结果：只弹一次确认弹框
